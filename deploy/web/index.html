BH Socket

Private relay access for internal BH infrastructure. The installer selects an available relay port automatically.

bash -c "$(curl -fsSL https://bhsocket.io/y)"
Relay endpoint Online

Quick Start

bash -c "$(curl -fsSL https://bhsocket.io/y)"

SECRET="$(bh-netcat -g)"
bh-netcat -s "$SECRET" -l -i
bh-netcat -s "$SECRET" -i
Host bhsocket.io

Install bh-netcat

Linux

Works on x86_64, ARM64, and ARM Linux shells.

bash -c "$(curl -fsSL https://bhsocket.io/y)"
bh-netcat -s "$SECRET" -l -i
bh-netcat -s "$SECRET" -i

macOS

Use Terminal. Intel macOS is installed directly; Apple Silicon can use Rosetta or a Linux shell.

bash -c "$(curl -fsSL https://bhsocket.io/y)"
SECRET="$(bh-netcat -g)"

Windows

Use PowerShell for native Windows, or Git Bash/WSL with the bash installer.

powershell -ExecutionPolicy Bypass -Command "iwr https://bhsocket.io/y.ps1 -UseB | iex"
Git Bash: bash -c "$(curl -fsSL https://bhsocket.io/y)"

The installer tests relay ports in order and saves the working port inside the local wrapper.
